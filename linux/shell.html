Linux: Shell

Main Command line

Shell is just another Linux program that provides the interface to Linux so you can run programs.

ash shell

ash shell, by Kenneth Almquist, is one of the smallest shells available for Linux. It supports most of the command shell commands.

bash shell

bash shell, or Bourne Again SHell, by Brian Fox and Chet Ramey, is the default Linux shell. It works just like the sh shell.

ksh shell

pdksh, or public-domain Korn shell, originally by Eric Gisin, supports job control.

tcsh shell

tcsh shell, by William Joy and others, emulates the csh shell, but has more features, i.e., a command-line editor with spelling correction.

zsh shell

zsh shell, originally by Paul Falstad, has more than 50 different command-line options and emulates the sh and ksh shell commands.

Shell Command Line

  • > : greater-than (>) character, a standard output redirection operator; the standard output has an assigned file number of one(1).
  • < : less-than (<) character, a standard input redirection operator; the standard input has an assigned file number of zero(0).
  • cpio -o < file1.txt >file1.cpio 2>cpio.errors : the standar error has an assigned file number of two(2)
    • <
  • <>> : concatenate (>>), append redirection operator
  • << : end of the input immediately follows the << operator

Customizing Shell

  • printenv : list the environment variables
  • add customized items into ~/.bash_profile

Running Programs in the Background

  • & : ampersand operator, for example, ls &
  • ps : process number, process status command
  • kill process_number : stop the program with its process number
  • Ctrl + z : put a running program into the background and suspend its operation
  • fg : bring the running program back to current display
  • bg : continue to allow the program to run in background
  • jobs : list the suspended programs
  • fg %1 or fg %programName : each job has a job number which can be refered, bring the first suspend program, for example.
  • kill %jobNumber or kill %programName : similar to fg %1 or fg %programName

Building Shell Commands

  • chmod +x filescript : turn the command lines into an executable file/script
